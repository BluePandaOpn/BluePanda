# Sprite2D

Visual helper methods for sprites.

## File

`BluePanda/Nodos/Sprite2d.py`

## Methods

- `flip_h(boolean)`
- `flip_v(boolean)`
- `set_opacity(alpha)`
- `set_scale(scale_x, scale_y)`
- `set_color(color)`
- `set_texture(path, width=None, height=None)`
- `tint(color, alpha=100)`

## Activation

Enabled via `@Sprite2D`.
