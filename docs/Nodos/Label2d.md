# Label2D (`Label2d.py`)

Nodo UI para renderizar texto en pantalla.

## Constructor

```python
Label2D(text="Texto", font_size=24, color=(255, 255, 255))
```

## Propiedades

- `text`
- `font_size`
- `font_color`
- `font_name`
- `is_ui = True`
- `pos`
- `image`
- `rect`

## Metodos

- `set_text(new_text)`
- `render_text()`
- `update()`

## Nota

Al tener `is_ui = True`, no recibe desplazamiento de camara.
