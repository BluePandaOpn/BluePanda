# AnimatedSprite2D

Frame animation using atlas/spritesheet textures.

`BluePanda/Nodos/AnimatedSprite2D.py`

## Methods

- `load_atlas(path, frame_width, frame_height, total_frames)`
- `update_animation()`
- `play()`
- `stop()`

With `@AnimatedSprite`, you can define:

- `Atlas`
- `frame_w`
- `frame_h`
