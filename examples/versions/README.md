python examples/versions/v02_player_move.py
python examples/versions/v03_collision_block.py
python examples/versions/v04_pickups_events.py
python examples/versions/v05_ui_button_counter.py

- `v01`: estructura minima.
- `v02`: movimiento con input.

- `v06`: camara siguiendo jugador.
- `v07`: grupos + pausa + time scale.
- `v08`: salud + patrulla enemiga.
- `v09`: emisor de particulas.
- `v10`: maquina de estados.
- `v11`: scheduler + tween.
- `v12`: showcase completo con sistemas combinados.
