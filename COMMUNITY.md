# Community

BluePanda is an open community project maintained by `Pato404`.

## How to participate

- Open issues for bugs and proposals.
- Submit pull requests with docs and tests when applicable.
- Share projects built with BluePanda.

## Recommended community structure for GitHub

- `Issues`: bugs and feature requests.
- `Discussions`: Q&A and ideas.
- `Wiki` or `docs/`: long-form guides.
